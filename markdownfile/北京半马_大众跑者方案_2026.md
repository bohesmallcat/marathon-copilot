## 二、官方赛道详解

### 完整路线

```
天安门广场(起点) → 东长安街 → 建国门内大街 → 建国门桥(右转) →
东二环辅路(北行) → 东四十条桥 → 东直门桥(左转) →
北二环辅路(西行) → 雍和宫桥 → 安定门桥(右转) →
安定门外大街(北行) → 鼓楼外大街 → 北辰路 →
大屯路 → 奥林匹克公园中心区景观大道(终点)
```

> 注：赛道路线以官方公布为准，上述路线基于北京半马经典线路。实际2026路线请参考官网公告。

### 赛道分段地形分析

| 路段 | 约公里数 | 地形特征 | 遮蔽 | 备注 |
|------|---------|---------|------|------|
| 天安门广场→东长安街 | 0-3km | 极平坦，超宽主干道 | 无遮蔽 | 长安街宽阔壮观，起跑人潮汹涌，需克制 |
| 建国门内大街→建国门桥 | 3-5km | 平坦 | 少量行道树 | 建国门桥右转进入东二环 |
| 东二环辅路（南→北） | 5-10km | 平坦，道路笔直 | 西侧有行道树 | 二环路段长且直，容易进入节奏；部分路段两侧高楼遮阳 |
| 北二环辅路（东→西） | 10-13km | 平坦 | 中等 | 雍和宫、钟鼓楼等历史地标在侧 |
| 安定门外大街→鼓楼外大街 | 13-16km | 平坦 | 中等行道树 | 穿越中轴线附近，人文氛围浓厚 |
| 北辰路→大屯路 | 16-19km | 平坦 | 较少 | 接近奥林匹克公园，道路开阔 |
| 奥林匹克公园景观大道 | 19-21km | 平坦 | 少 | 鸟巢、水立方映入眼帘，冲刺路段 |

### 赛道关键特征

1. **点对点赛道，全程向北**：从天安门向北直通奥体中心，整体方向明确，无折返。心理上"一路向前"非常友好
2. **极度平坦**：北京城区几乎零爬升，全程海拔差不到 10m，是国内最快的半马赛道之一
3. **超宽赛道**：长安街和二环辅路路面极宽（6-8 车道），拥挤度远低于一般城市赛道
4. **城市峡谷效应**：二环路段两侧高楼形成"城市峡谷"，可能产生局部风道或遮阳效果
5. **地标密集**：天安门→建国门→东直门→雍和宫→鼓楼→鸟巢，全程顶级城市景观，但也容易分心
6. **无折返损耗**：点对点赛道没有折返减速/加速的时间浪费

## 五、赛道分段策略

### 第一段：0-5km —「天安门起步，克制出发」
**路段**：天安门广场 → 东长安街 → 建国门内大街 → 建国门桥

- **配速**：比目标均配慢 5-8"/km
- **心率**：控制在最大心率 75-80%
- **要点**：
  - 天安门广场起跑气势恢宏，肾上腺素飙升，**最容易冲快的 3km**
  - 长安街极度宽阔平坦，没有任何地形制约，更需要自制力
  - 07:00 气温 10-12°C，肌肉尚在激活阶段，起步偏慢是对的
  - 建国门桥右转进入东二环，注意弯道处的人流汇聚
  - **黄金法则：前 3km 心率不超过 80%，配速感觉"轻松偏慢"才是对的**

### 第二段：5-10km —「二环巡航，锁定节奏」
**路段**：东二环辅路（建国门桥 → 东四十条桥 → 东直门桥）

- **配速**：切入目标均配
- **心率**：最大心率 80-85%
- **要点**：
  - 二环辅路又直又平，是建立稳定节奏的理想路段
  - 路两侧高楼形成城市峡谷，上午阳光从东侧照射，西侧辅路可能有阴影
  - 这是全程最"无聊"的路段，但也是最关键的效率段——**稳定输出，不追不抢**
  - 8km 左右体温升至工作状态，可能开始感觉手臂发热——这是正常信号
  - 注意二环路面状况，辅路可能有公交站台凸起或路面接缝

### 第三段：10-15km —「北二环→中轴线，稳步推进」
**路段**：北二环辅路 → 安定门外大街 → 鼓楼外大街

- **配速**：保持目标均配
- **心率**：最大心率 83-87%
- **要点**：
  - 东直门桥左转进入北二环，方向由北转西再转北，弯道节奏不要乱
  - 安定门桥右转向北，进入安定门外大街→鼓楼外大街，穿越北京中轴线
  - 两侧有雍和宫、钟鼓楼等地标，观众密集，加油声助推
  - 13-15km 是半马的"中段关卡"——体感从舒适开始转向微疲劳，**主动调整呼吸节奏**
  - 此时气温升至 15-17°C，体感最佳，是巩固配速的好时机

### 第四段：15-19km —「北辰路，直面阳光」
**路段**：鼓楼外大街 → 北辰路 → 大屯路

- **配速**：保持目标均配，体感好可开始微提
- **心率**：最大心率 86-89%
- **要点**：
  - 进入北辰路后道路开阔，遮蔽减少，阳光直射感明显增强
  - 16-18km 是半马的"撞墙区"，糖原消耗加大，配速容易无意识下滑
  - 15km 处及时补充能量胶，为最后 5km 储备
  - 远处开始看见鸟巢轮廓——**视觉目标是强大的心理推动力**
  - ⚠️ 气温升至 18-20°C，低湿度下可能口渴感不明显但仍在脱水。**主动补水**

### 第五段：19-21.1km —「奥体冲线，全力释放」
**路段**：大屯路 → 奥林匹克公园中心区景观大道（终点）

- **配速**：体感好就提速 10-15"/km 冲刺；体感一般维持均配
- **心率**：可达最大心率 90-93%
- **要点**：
  - 进入奥林匹克公园区域，鸟巢和水立方近在咫尺，终点氛围感极强
  - 这是你前 18km 克制积攒的红利时刻——**前面省下的体能在这里兑现**
  - 景观大道平坦宽阔，适合全力释放
  - 最后 500m 终点拱门在视线内，如有余力可全力冲刺
  - **熔断规则**：心率 92%+ 且配速在掉 → 不硬拉，平稳跑完最后 2km

## 六、补给策略

### 赛前

| 时间 | 补给内容 |
|------|---------|
| 赛前 3 小时（04:00） | 早餐：白米饭/面包 + 果酱（高碳水低纤维），200ml 温水 |
| 赛前 1 小时（06:00） | 100-150mg 咖啡因（可选，需赛前已验证耐受性） |
| 赛前 10 分钟（06:50） | 能量胶 1 支 + 100ml 水 |

### 赛中

| 公里数 | 补给内容 |
|--------|---------|
| 每个补给站 | 2-3 口水或运动饮料（约 100-150ml） |
| 10km（北二环段） | 能量胶 1 支 |
| 15km（北辰路前） | 2-3 口运动饮料 + 视体感决定是否补胶 |

### 晴天干燥补水要点

- **北京春季空气干燥（湿度 30-40%）**，汗液蒸发极快，皮肤表面感觉"干爽"，但实际脱水速度可能比湿热天气更快
- **口渴感滞后于脱水**：干燥环境下不口渴 ≠ 不缺水。每个补给站都应至少喝 2-3 口
- 低湿度下鼻腔和咽喉容易干燥，跑前可涂少量凡士林于鼻腔内壁
- 电解质同样重要——干燥天气出汗蒸发快，盐分流失不容易被察觉

## 七、装备建议

### 穿着建议（10-20°C，晴天干燥，紫外线强）

| 部位 | 建议 |
|------|------|

| 下装 | 短裤。气温舒适，短裤是最佳选择 |
| 鞋 | 竞速跑鞋（碳板鞋在此平坦赛道上可充分发挥优势）。晴天路面干燥，抓地力无忧。勿穿新鞋 |
| 帽子 | **遮阳帽/空顶帽（强烈推荐）**：晴天紫外线"很强"，全程几乎无遮蔽 |
| 袜子 | 轻薄速干跑步袜 |
| 防晒 | **SPF30+ 防晒霜**：涂抹面部、颈部、手臂、小腿。赛前 20 分钟涂好，选防汗配方 |
| 眼镜 | 运动太阳镜（可选，晴天上午太阳偏东，部分路段可能直射） |

### 晴天特别装备

- **一次性保暖层（推荐）**：起跑前天安门广场等候时间长且气温仅 10°C，一件旧T恤或垃圾袋套住上身保温，起跑后丢弃
- **防晒霜（必备）**：4月北京紫外线已很强，半马 1.5-2.5 小时的日晒足以晒伤
- **防花粉口罩/药物（过敏体质必备）**：4 月北京花粉浓度极高，过敏性鼻炎跑者赛前服用抗组胺药

## 九、风险提示与熔断规则

### 分级熔断机制

| 信号 | 触发条件 | 应对措施 |
|------|---------|---------|
| 🟢 绿灯 | 配速/心率均在计划内，体感舒适 | 继续执行计划 |
| 🟡 黄灯 | 前半程心率超预期 5-8 次/分，或后半程体感明显发热 | 降配速 5-8"/km，增加补水频次 |
| 🟠 橙灯 | 连续 3km 心率超限 或 配速掉 15"/km+ | 放弃时间目标，切换「安全完赛」模式 |
| 🔴 红灯 | 头晕/恶心/中暑前兆/视线模糊/抽筋不止 | **立即停跑，求助医疗点** |

### 本场赛事特别注意

1. **天安门起跑不冲动**：壮观的场面 + 万人起跑的肾上腺素是最大的"配速杀手"。前 3km 如果比计划快 15 秒以上，后面一定会付出代价
2. **干燥环境隐性脱水**：北京春季湿度 30-40%，汗液蒸发极快，体表感觉干爽但实际在快速脱水。**不渴也要喝，每站必补**
3. **后半程日晒升温**：8:30 后气温快速升至 18-20°C+，加上紫外线直射。16km 后如感觉体温明显上升、心率漂移，及时降速
4. **花粉过敏**：4 月北京花粉浓度极高（气象网花粉指数"很高"）。有过敏史的跑者若出现严重鼻塞/呼吸困难，不要硬撑
5. **赛后注意保暖**：完赛后身处奥林匹克公园开阔区域，虽然气温 20°C 但停跑后体表蒸发散热，穿上干衣避免着凉
6. **人流管理**：北京半马参赛人数庞大，起跑和补给站可能拥挤。补给站提前减速，拿水后移至外侧饮用，不要急停

| 时间 | 内容 |
