```
1km 跑 → SkiErg 1000m → 1km 跑 → Sled Push 2×25m →

1km 跑 → Rowing 1000m → 1km 跑 → Farmers Carry 200m →
1km 跑 → Sandbag Lunges 100m → 1km 跑 → Wall Balls 100rep
```

| 项目 | 计算方法 | 示例（半马 PB 1:40） |

## 2. 发力模式纠正——从跑步到 HYROX

大众跑者最大的适应障碍不是体能，而是**发力模式切换**。跑步用弹性反弹，HYROX 站点用刚性输出——这是两套完全不同的神经肌肉模式。

### 纠正 #1: 足部发力 — 前掌弹性 → 全掌蹬地

| | 跑步模式 | HYROX 站点模式 |
| :--- | :--- | :--- |
| 着地 | 前掌/中掌弹性回弹 | 全掌踩实，脚跟不离地 |
| 力来源 | 跟腱弹性能 + 小腿 | 股四头 + 臀肌直接蹬地反作用力 |
| 适用站 | — | Sled Push, Sled Pull, Wall Balls, Lunges |

**训练动作：** Box Squat 3×8（全蹲起立，脚跟蹬地）

### 纠正 #2: 髋关节功能 — 摆动节奏 → 扭矩输出

| | 跑步模式 | HYROX 站点模式 |
| :--- | :--- | :--- |
| 髋功能 | 小幅度高频摆动 | 大幅度髋伸展 → 产生推拉力 |
| 适用站 | — | SkiErg, Rowing, Sled Pull, Kettlebell Swing |

**训练动作：** Kettlebell Swing 3×12-15（男 20-24kg / 女 12-16kg）

### 纠正 #3: 核心功能 — 弹性扭转 → 刚性支撑

| | 跑步模式 | HYROX 站点模式 |
| :--- | :--- | :--- |
| 核心功能 | 轻微旋转传递能量 | 刚性支撑抗侧屈/抗旋转 |
| 适用站 | — | Sled Push, Farmers Carry, Wall Balls |

**训练动作：** Pallof Press 3×12 + 单臂 Farmers Carry 3×50m/侧

### 纠正 #4: Sled Push 专项技术（全场最难站）

Sled Push 是绝大多数大众跑者的"噩梦站"——152kg（男）或 102kg（女）的雪橇，对 60-80kg 的跑者来说是极大的挑战。

**关键技术要点：**

| 要素 | 标准 |
| :--- | :--- |
| 身体角度 | **35-45°**（体重越轻角度越低，增加水平推力分量） |
| 步频 | 高频短碎步（每步 15-30cm），不追求大步幅 |
| 启动 | 前 3-5 步爆发突破静摩擦，之后锁定节奏 |
| 呼吸 | 每 3-5 步呼吸一次 |
| 核心原则 | **永远不要停下来。** 一旦停止，重新启动的代价远大于慢速持续推进 |
| 鞋底 | **底纹最深的训练鞋**（绝不穿竞速跑鞋，抓地力 = 生命） |

## 4. 比赛执行策略

### 前半程 (站 1-4): 控制。80% 努力。

- **1km 跑**：感觉应该是"可以再快但我选择不快"
- **SkiErg (站 1)**：用髋驱动，省手臂力量给后面的 Sled Push
- **Sled Push (站 2)**：🔥 **关键站。** 极低角度，短碎步，永远不停。这是全场最难的站——平稳完成就是胜利
- **Sled Pull (站 3)**：坐低拉，用腿和背发力，不纯拉手臂
- **Burpee Broad Jump (站 4)**：节奏稳定 > 拼速度。每跳 1.2-1.8m

### 转换区 (Roxzone)

- **每站出来后步行 20-30m，深呼吸 3-5 次**
- 走 20m 省下的心率恢复 > 跑 20m 节省的时间
- Sled Push 出站后多走 10m（共 30-40m），让心率从峰值下降再起跑

### 后半程 (站 5-8): 释放。90% 努力。

- **Rowing (站 5)**：腿→髋→臂，节奏稳定
- **Farmers Carry (站 6)**：挺胸不停不放，200m 一气呵成
- **Sandbag Lunges (站 7)**：控制步幅，膝盖轻触地。**节奏稳 > 冲快**
- **Wall Balls (站 8)**：最后一站，全力释放。分组 20-25rep，组间站立休息 10-15"

### 最后 1km 跑

**释放全部储备。冲到终点。**

### 比赛中补给

| 时间点 | 补给 |
| :--- | :--- |
| 赛前 30min | 能量胶 1 支 + 咖啡因 100-150mg |
| 赛中 ~30-35min (Rowing 后) | 200ml 运动饮料（利用 Rowing 站后转换区） |
| 赛中 ~55min (Lunges 后) | 能量胶 1 支（如预计总时间 > 75min） |

```
  | W4 | — | —% | 目标时间内？ |
```

### KPI #2: 受干扰跑步恢复速度 (CR Recovery)

```
测量方法：力量站完成后立即起跑，记录前 400m 的平均心率和配速
评估标准：
  优秀：400m 内 HR 下降 ≥ 15bpm，配速达到目标 CR
  良好：400m 内 HR 下降 10-15bpm，配速比目标慢 ≤10"/km
  需改进：400m 内 HR 下降 < 10bpm，配速比目标慢 > 15"/km

重点站后数据：Sled Push 后的 CR Recovery — 这是最难恢复的站
```

### KPI #3: 全程模拟 Pacing 一致性

```
测量方法：8 段 1km 跑步配速的标准差
评估标准：
  优秀：标准差 < 10"/km (配速均匀)
  良好：标准差 10-15"/km
  需改进：标准差 > 15"/km (前快后崩)

大众跑者常见陷阱：前 3 段 1km 跑太快 → 后半程崩盘
```

### KPI #4: 过度训练预警

| 信号 | 行动 |
| :--- | :--- |
| 晨脉 > 基线 + 5bpm 连续 2 天 | 当日改为恢复训练 |

## 6. 场馆与环境策略

### 6.1 室内场馆环境预估

| 因素 | 预估 | 对策 |
| :--- | :--- | :--- |
| **场馆类型** | 室内展览馆 | 密闭空间，温度/湿度较室外高 |
| **室温** | 25-30°C（5 月 + 室内人群热量） | 轻薄速干短袖 + 短裤 |
| **湿度** | 60-80% | 赛前赛中增加补水频率 |
| **空气流通** | 较差 | 控制呼吸节奏。转换区刻意深呼吸 3-5 次 |
| **地面** | 人工地面/橡胶地面 | **极重要！** 鞋底抓地力 = Sled Push 成败 |
| **噪音** | 极大（DJ 音乐 + 观众） | 依赖手表/心率监控配速，不依赖体感 |

### 6.2 装备清单

| 装备 | 推荐 | 说明 |
| :--- | :--- | :--- |
| **鞋** | **底纹最深的训练鞋** | 抓地力决定 Sled Push 成败。**绝不穿竞速跑鞋/气垫鞋** |
| **手套** | 推荐（Sled Pull / Farmers Carry） | 减少握力损耗，防手掌磨破 |
| **上衣** | 轻薄速干短袖 | 室内闷热，越轻越好 |
| **裤** | 短裤 + 压缩内裤 | 防磨（Burpee + Lunge） |
| **手表** | 跑步手表 | 记录心率 + 分段计时 |
| **防磨** | 凡士林/Body Glide | 内侧大腿 + 腋下 |
| **补给** | 能量胶 ×1-2 + 运动饮料 | 贴号码布内侧或放口袋 |

### 6.3 鞋底抓地力测试建议

Sled Push 对跑者来说，鞋底选择的影响远大于你的想象——体重越轻影响越大。

**赛前建议：** 用 2-3 双不同训练鞋分别推比赛重量雪橇 25m，计时对比。选择时间最快的那双。
- 综合训练鞋（Nike Metcon / Reebok Nano / NO BULL）通常抓地最好
- **绝对避免**：光滑底跑鞋、气垫鞋（推力被气垫吸收）、高帮鞋（限制踝关节活动）

## 7. 风险管理

### 7.1 Sled Push 失败预案

Sled Push 是大多数大众跑者 HYROX 中确定性最低的站。必须准备预案：

| 场景 | 信号 | 应对 |
| :--- | :--- | :--- |
| **正常** | 每 25m 按目标完成，连续不停 | 继续执行配速计划 |
| **困难但可控** | 需要停 1-2 次/被反弹 | 降低预期。后续跑步段不要加速补偿 |
| **严重困难** | 每 25m > 3:30，频繁停下 | 放弃时间目标。80% 努力推完即可 |
| **地面打滑** | 鞋底抓不住 | 更低角度增加垂直力分量。极端情况可脱鞋赤脚 |

### 7.2 配速心态管理

大众跑者参加 HYROX 最常见的失误：

| 陷阱 | 表现 | 预防 |
| :--- | :--- | :--- |
| **前半程冲太猛** | 前 3 站全力 → 后 5 站崩盘 | 前半程 ≤85% 努力。**"还能更快"才是对的** |
| **跳过转换区恢复** | 出站立即冲跑 | 每站出来走 20-30m，这不是跑步比赛 |
| **站点拼到力竭** | Wall Ball 冲 40rep 然后趴下 | 分组策略（20-25rep/组）。永远不力竭 |
| **忽视跑步配速** | 跑步段不看手表 | 每段 1km 必须看表。超速或掉速 > 10" → 主动调整 |

## 8. 总结

### 6 周训练要点回顾

| 周次 | 重点 | 核心指标 |
| :--- | :--- | :--- |
| W1 | 动作学习 + 基线测试 | 记录所有站点和 CR 基线时间 |
| W2 | 上全比赛重量 | Sled Push/Pull 首次 100% 重量 |
| W3 | 首次全程模拟 | 总时间 + 配速标准差 + 最弱站 |
| W4 | 弱点突破 + 第二次全程 | 比 W3 快 2-3 分钟 |
| W5 | 减量 50% | 保持锐利感，装备准备 |
| W6 | 比赛周 | 执行赛日流程 |

### 大众跑者 HYROX 成功三原则

1. **Sled Push 决定一切**——把 60% 的专项训练时间投资在 Sled Push 上。其他站点凭跑步/力量底子适应即可
2. **均匀输出 > 爆发冲刺**——HYROX 奖励的是稳定配速，不是某一段的极速。出站走 20m 再跑，每次都看表
3. **鞋底抓地力是装备第一优先级**——鞋选错，所有 Sled Push 训练白费

> **享受你的第一场 HYROX。** 这是一项独特的比赛体验——跑步和力量的交替切换会带来前所未有的挑战感和成就感。不管目标时间是多少，跑过终点线的那一刻你会知道：值得。
