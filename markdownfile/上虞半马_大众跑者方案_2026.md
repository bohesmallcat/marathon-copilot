## 二、官方赛道详解

### 完整路线

```
上虞体育中心西门(起点) → 曙兴路(直行) → 江东北路(左转) →
市民大道(左转) → 市民大道与峰山南路交界处(折返①) → 市民大道(直行) →
江东北路(右转) → 便民中心内部路(右转) → 三环北桥辅道(左转) →
复兴路(左转) → 鸿雁路(左转) → 江西路(右转) →
一江两岸景区(左转) → 五甲渡大桥(右转) → 一江两岸景区(右转) →
江东北路(左转·折返②) → 曙兴路(左转) → 上虞体育中心西门(终点)
```

### 赛道分段地形分析

| 路段 | 约公里数 | 地形特征 | 遮蔽 | 备注 |
|------|---------|---------|------|------|
| 体育中心→曙兴路→江东北路 | 0-2km | 平坦宽阔城市道路 | 少量行道树 | 起跑人流密集，需克制配速 |
| 江东北路→市民大道 | 2-5km | 平坦，城市主干道 | 较少 | 市民大道宽阔，侧风无遮挡 |
| 市民大道折返①→返回江东北路 | 5-8km | 平坦 | 较少 | 折返点处需减速转向，回程同路段 |
| 江东北路→便民中心→三环北桥辅道 | 8-10km | 平坦 | 中等 | 便民中心内部弯道较多，路面较窄 |
| 复兴路→鸿雁路→江西路 | 10-13km | 平坦，城市道路 | 中等 | 过渡路段，逐渐接近曹娥江滨 |
| **一江两岸景区→五甲渡大桥** | **13-17km** | **滨江步道 + 桥梁爬升** | **开阔无遮蔽** | 景区路段风景最佳，桥梁有短爬升；**沿江段风力最大** |
| 五甲渡大桥→一江两岸景区返程 | 17-19km | 滨江步道，平坦 | 开阔 | 下桥后顺沿江道返回 |
| 江东北路折返②→曙兴路→终点 | 19-21km | 平坦城市道路 | 少量行道树 | 回到起点区域，最后冲刺 |

### 赛道关键特征

1. **整体极度平坦**：上虞城区地势平坦，全程几乎无爬升，仅五甲渡大桥有约 10-15m 短暂上坡。理论上是"快赛道"，但中雨天气大幅抵消了地形优势
2. **2 处折返**：市民大道折返（约 5km 处）和江东北路折返（约 19km 处），每个折返点减速/转向损耗约 3-5 秒
3. **滨江开阔段约 6km（13-19km）**：一江两岸景区沿曹娥江，无建筑遮挡，是风力和雨势影响最大的路段
4. **弯道/转向多**：全程含约 16 次转弯（左转 8 次、右转 5 次、折返 2 次、直行穿插），雨天路面湿滑时需格外注意弯道
5. **实际跑动距离修正**：因弯道切线和折返损耗，建议按 **21.30km** 估算体能消耗

### 天气评价

**降雨（最大不利因素）**：预报全天中雨，这意味着从起跑到完赛**全程在雨中进行**。中雨对比赛的影响是全方位的——路面湿滑、鞋袜进水增重、视线受阻、体感温度下降、心理压力增大。**本场比赛的核心命题是"如何在雨中安全且高效地完赛"**。

**温度（中性偏有利）**：14-21°C 本身处于半马适宜区间，但中雨 + 高湿度使得体表散热通道从"蒸发"转为"淋湿后风冷"。前半程体感偏凉，后半程随温度升至 18°C+ 可能出现"闷热+淋雨"的矛盾体感。

**湿度（极不利）**：中雨天气湿度 90%+，排汗几乎无法蒸发。身体通过出汗散热的效率大幅下降，加上雨水淋湿衣物贴身，实际热负荷高于体感温度暗示的水平。

**风力（有利）**：<3 级弱风是唯一的利好因素。风阻极小，但沿江开阔段（13-19km）即使弱风也会因无遮挡而感受更明显。

## 五、赛道分段策略

### 第一段：0-5km —「雨中热身，耐心起步」
**路段**：体育中心 → 曙兴路 → 江东北路 → 市民大道

- **配速**：比目标均配慢 10-15"/km
- **心率**：控制在最大心率 72-78%
- **要点**：
  - 15000 人同时起跑，开局必然拥挤。雨天路面湿滑，起跑区更需耐心
  - 曙兴路和市民大道宽阔平坦，容易兴奋冲快，**雨天前 3km 必须比晴天更保守**
  - 起跑气温约 15-16°C + 淋雨，体感偏凉，肌肉激活更慢
  - 雨中视线不佳，注意脚下路面积水和井盖
  - **黄金法则：雨战首 5km 以"控制"为核心，前面省下来的都是后面的本钱**

### 第二段：5-10km —「折返稳速，建立节奏」
**路段**：市民大道折返 → 江东北路 → 便民中心 → 三环北桥辅道

- **配速**：逐步切入目标均配（已下调后的目标）
- **心率**：最大心率 78-83%
- **要点**：
  - 5km 处市民大道折返，**提前 50m 减速，弯心不急转**。湿滑路面急转弯极易滑倒
  - 折返后回程路段与来时相同，心理上不新鲜但地形熟悉
  - 便民中心内部路面较窄、弯道多，降雨天格外注意路面标线（标线湿后极滑）
  - 到这里身体应完成热身，雨衣若还穿着、体感开始闷热的话可以丢弃
  - 此阶段核心任务：**在湿润体感中找到可持续的舒适节奏**

### 第三段：10-14km —「城市过渡，蓄力滨江」
**路段**：复兴路 → 鸿雁路 → 江西路 → 一江两岸景区入口

- **配速**：保持稳定目标均配
- **心率**：最大心率 83-86%
- **要点**：
  - 复兴路→鸿雁路→江西路为城市道路过渡段，有建筑物部分遮风挡雨
  - 10km 处补给能量胶——**雨天用手撕胶时手滑，建议提前撕开一个口或使用易撕包装**
  - 临近 13km 进入一江两岸景区前，做好心理准备：即将进入全程最开阔、风雨最大的路段
  - 半马的"中段疲劳关"通常在 12-14km，叠加雨战心理消耗，此时需**主动自我激励**
  - 雨天跑 1 小时后鞋袜已完全湿透——这是正常的，不要因此改变跑姿

### 第四段：14-18km —「滨江雨战，核心路段」⚠️ 关键路段
**路段**：一江两岸景区 → **五甲渡大桥** → 一江两岸景区（返程）

- **配速**：允许掉 5-10"/km，上桥段不追速
- **心率**：最大心率 85-88%
- **要点**：
  - **这是全场最困难的 4km**：沿曹娥江的滨江道完全开阔，无遮挡，风雨直面
  - 五甲渡大桥有约 10-15m 短爬升，桥面风力明显增大且**桥面湿滑风险最高**
  - ⚠️ **上桥保心率不保配速**，下桥不冲刺——桥面下坡 + 湿滑 + 弯道是摔倒高发组合
  - 沿江段温度体感比城市段低 1-2°C（水面降温+风冷效应）
  - 过了五甲渡大桥就进入"回家之路"，**咬住这 4km 就是胜利**

### 第五段：18-21.1km —「回程收官，量力冲刺」
**路段**：江东北路折返 → 曙兴路 → 体育中心终点

- **配速**：体感好可恢复至目标配速或提 5"/km；体感差就保持匀速
- **心率**：可达最大心率 88-90%
- **要点**：
  - 19km 处江东北路折返，同样**提前减速，湿滑弯道小心**
  - 回到曙兴路意味着接近终点，回程路段有观众加油
  - 最后 2km 回到起点区域，如果前 18km 控制得好，这里应有余力小幅提速
  - **熔断规则**：心率持续 90%+ 且配速仍在下掉 → 不硬拉，匀速安全跑完
  - 终点在体育中心西门，注意终点前路面可能因人流聚集更湿滑

## 六、补给策略

### 赛前

| 时间 | 补给内容 |
|------|---------|
| 赛前 3 小时（04:30） | 早餐：白米饭/面包 + 果酱（高碳水低纤维），200ml 温水 |
| 赛前 1 小时（06:30） | 100-150mg 咖啡因（可选，需赛前已验证耐受性） |
| 赛前 10 分钟（07:20） | 能量胶 1 支 + 100ml 水 |

### 赛中

| 公里数 | 补给内容 |
|--------|---------|
| 每个补给站 | 2-3 口水或运动饮料（约 100-150ml） |
| 10km（进入一江两岸景区前） | 能量胶 1 支 — 为后续滨江风雨段储备能量 |
| 15km（五甲渡大桥后） | 2-3 口运动饮料 + 视体感决定是否补胶 |

### 雨战补给特别要点

- **淋雨 ≠ 不脱水**：最常见的雨战误区。全身被雨水淋湿会产生"不缺水"的错觉，但身体仍在通过出汗和呼吸持续脱水。**每个补给站都要喝**
- **电解质优先于纯水**：雨天体表散热不畅，出汗量可能比体感暗示的更大。运动饮料比纯水更重要
- **能量胶要防水**：提前将能量胶放在防水腰包或贴身口袋中。雨天手滑难撕开，赛前可预先撕开一角
- **避免喝到雨水稀释的运动饮料**：补给站纸杯若被雨水稀释，补给效果打折。如可能，优先拿密封瓶装饮料

## 七、装备建议

### 穿着建议（15-19°C，全程中雨，高湿）

| 部位 | 建议 |
|------|------|
| 上衣 | 短袖速干 T 恤（深色优先，雨天浅色湿透后可能透视）。**起跑必穿一次性雨衣**（保温+防雨），跑至 5-8km 体热后丢弃 |
| 下装 | 短裤。高湿雨天散热优先，不建议压缩裤。**短裤内衬防磨处理** |
| 鞋 | 竞速鞋或训练鞋，**抓地力是第一考量**。避免纯碳板薄底鞋（湿滑路面抓地差）。勿穿新鞋 |
| 帽子 | **带帽檐的跑步帽（必备中的必备）**：中雨天的帽檐是唯一能挡雨水入眼的装备 |
| 袜子 | 速干薄款跑步袜。可考虑羊毛混纺袜（湿后仍保暖且防磨）。雨天鞋袜必湿，**接受它** |
| 防护 | **大量涂抹凡士林/Body Glide**：大腿内侧、腋下、乳头、脚趾、脚后跟。雨水会洗掉部分防磨膏，建议涂厚一些 |
| 眼镜 | 运动眼镜/透明镜片（挡雨滴入眼）。近视跑者戴隐形优于框架眼镜 |

### 雨战特别装备

- **一次性雨衣 x2（强烈推荐）**：一件赛前等候 + 起跑穿，一件存包袋里赛后穿。中雨天赛后失温风险极高
- **大号透明塑料袋 x2**：一个装手机，一个存放赛后干衣物防淋湿
- **备用干衣物全套（必备）**：干内衣、干T恤、干袜子、干鞋（如有），存包袋中。赛后第一时间换干衣
- **防水腰包/臂包**：装手机、能量胶、零钱等
- **乳贴/创可贴**：湿衣服反复摩擦，乳头磨伤概率远高于晴天

## 八、热身方案

| 时间 | 内容 |
|------|------|
| 07:00（起跑前30分钟） | 到达起点，**穿好一次性雨衣**，在遮蔽处（体育中心室内/走廊）等候 |
| 07:10（起跑前20分钟） | 原地小幅活动：踮脚、高抬腿、开合跳（不建议雨中户外慢跑热身，避免提前湿透） |
| 07:15（起跑前15分钟） | 动态拉伸：高抬腿、踢臀跑、侧滑步各 20m x 2（在遮蔽区完成） |
| 07:22（起跑前8分钟） | 2-3 次 50m 渐加速跑（加到比赛配速即可）。**如果雨大，省略此步，用前 2km 当热身** |
| 07:27（起跑前3分钟） | 进入出发区，深呼吸，雨衣保持穿着 |
| 07:30 | 鸣枪起跑 |

> 中雨 + 15°C：**核心原则是赛前尽量保持干燥和体温**。热身可以简化，用赛道前 2-3km 替代，比起跑前淋湿要明智得多。雨衣穿到跑热为止，不要过早丢弃。

## 九、风险提示与熔断规则

### 分级熔断机制

| 信号 | 触发条件 | 应对措施 |
|------|---------|---------|
| 🟢 绿灯 | 配速/心率均在计划内，雨中体感可控 | 继续执行计划 |
| 🟡 黄灯 | 前半程心率超预期 5-8 次/分，或体感开始发冷/过热 | 降配速 10"/km，观察 2km |
| 🟠 橙灯 | 连续 3km 心率超限 或 配速掉 15"/km+ 或 出现打滑/不稳 | 放弃时间目标，切换「安全完赛」模式 |
| 🔴 红灯 | 头晕/恶心/失温发抖/视线模糊/抽筋不止/滑倒受伤 | **立即停跑，求助医疗点** |

### 本场赛事特别注意

1. **雨天路面湿滑是第一风险**：所有弯道（16 次转弯 + 2 处折返）都需提前减速。路面标线、井盖、金属排水盖板湿后极滑，**主动绕开**
